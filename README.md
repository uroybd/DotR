# DotR
A dotfiles manager that is as dear as a daughter.

## Usage

```
Usage: dotr [OPTIONS] [COMMAND]

Commands:
  init    Initialize the application
  import
  copy
  update
  help    Print this message or the help of the given subcommand(s)

Options:
  -w, --working-dir <WORKING_DIR>
  -h, --help                       Print help
```
